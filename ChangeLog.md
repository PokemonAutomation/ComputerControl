**Version 0.68.9 (beta):**

- CC will now attempt to use the NV12 video format by default.
- Added a dropdown to select the video format (which allows restoring the old video behavior).
- Fixed SwSh Egg Autonomous getting stuck. (reported by many people)
- SV Egg Hatcher/Autonomous will now attempt to kill the wild Pokemon if you are attacked. (reported by many people)
- Added Pokopia Daily Farmer. (credit Gimikyu)
- FRLG RNG for Unown. (credit Astro)
- Added game icons to the game select dropdown.
- Donut Maker will now attempt to recover from certain kinds of errors.
- Improve reliability of Pokopia programs. (reported by Keagan, credit dolphincurry)
- Improve reliability of game re-entry. (reported by Butters)
- Improve reliability of LGPE battle programs. (reported by Lairette)
- Fixed OCR issues. (reported by Drak3, credit kichi + jw)
- Improve reliability of Pokopia program. (reported by Keagan, credit Gimikyu)
- Removed firmware for Arduino Uno R3, Arduino Leonardo, Teensy, and Pro Micro.

The video format change is potentially big here as it will effectively double the capture-card frame rate for many users. For those of you with the Switch 2 and a 4k capture card, pay attention to see if you are finally able to achieve 4k @ 60 FPS. Please let us know if you encounter any issues. There are more video options that we added which are hidden by default.
