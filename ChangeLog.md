**Version 0.61.4:**

- Donut Maker can now make multiple donuts. (credit Gimikyu)
- Improved reliability of Donut Maker. (credit Gin, Deader)
- Improved reliability of Fly Spot Reset in hyperspace. (from error reports)
- Improved reliability of Mega Shard Farmer. (reported by ArceusMI)
- If you try to use an OEM controller without enabling it in the Switch settings, you will now get an error message*.
- Added alternate keyboard mappings for the Dpad for keyboards without a numpad. You will need to reset your keyboard mappings back to default to pick this up. (credit Saber)
- The old deprecated Sword/Shield programs have been removed.

*Requires flashing new firmware. Old firmware will not show this message.
